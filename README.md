# Multipass redis VM
This project help to create an instance of redis (with redis-stack) in a multipass vm
## Install VM
### Set parameters:
```
cp env.dist env
```
### Edit env parameters
```
nano env
```
### Install
```
./install.sh
```
### Uninstall
```
./uninstall.sh
```
